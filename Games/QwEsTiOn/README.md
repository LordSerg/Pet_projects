# QWESTION

## Technology used
C#, WindowsForms, Math, VisualStudio

## Overview
Technically it's hard to call it a **game**, it's rather quest, or a riddle. This project was made for [Acerola Jam 0](https://itch.io/jam/acerola-jam-0). 

## About project
